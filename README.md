# 🔐 JWT Authentication API

## 📌 Description
This project is a REST API built using Java for handling user authentication. It includes login functionality and JWT-based authentication for secure access to protected routes.

## 🚀 Features
- User Login Authentication
- JWT Token Generation
- Secure API Access using Token
- RESTful API Structure

## 🛠️ Tech Stack
- Java
- Spring Boot (if used, otherwise remove)
- REST API
- JWT

## 📡 API Endpoints
- POST /login → User login
- GET /protected → Access secured route

## ▶️ How to Run
1. Clone the repository
2. Open in IDE (IntelliJ / VS Code)
3. Run the application
4. Test APIs using Postman

- Understood authentication flow
- Implemented JWT-based security
